# 🎯 SoloCTO Prompt Kit

> **AI Rules Pack** — всё что нужно, чтобы AI работал по-твоему.
> Установка: 5 минут. Результат: AI превращается из помощника в напарника.

## Что внутри

### 📁 `.agent/rules/` — 6 файлов правил
Просто скопируй папку в корень проекта. AI (Cursor, Copilot, Gemini, Claude) будет читать эти правила автоматически.

### 📋 Шаблоны
- `DECISIONS.md` — журнал архитектурных решений
- `SOLUTION_PATTERNS.md` — библиотека проверенных паттернов
- `CHANGELOG.md` — структурированный лог изменений

### 🧪 Code Complexity Levels
- Автоматическая классификация кода по 3 уровням сложности
- AI сам решает сколько тестов нужно
- Post-mortem протокол при проскочивших багах

### 📖 Docs Template (Documentation 2.0)
- `1_PRODUCT.md` — что строим и для кого
- `2_ARCHITECTURE.md` — как устроено
- `3_DEPLOYMENT.md` — как запускать
- `4_DECISIONS.md` — почему так

## Быстрый старт

```bash
# 1. Скопируй в свой проект
cp -r solocto-prompt-kit/.agent your-project/.agent
cp solocto-prompt-kit/templates/* your-project/

# 2. Заполни MAKER_PROFILE.md (5 мин)
# Открой .agent/rules/MAKER_PROFILE.md и ответь на вопросы

# 3. Всё! AI теперь знает твои правила.
```
