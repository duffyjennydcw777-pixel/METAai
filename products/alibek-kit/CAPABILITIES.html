💡 Примеры что можно сделать

💬 Пример 1: Создать лендинг
Ты: "Создай красивый лендинг для магазина кроссовок. Тёмная тема, неоновые акценты, анимации при скролле." AI: *создаёт полный HTML/CSS/JS файл с работающим сайтом*
💬 Пример 2: Telegram бот
Ты: "Сделай Telegram бота, который отвечает на вопросы по моему прайс-листу" AI: *создаёт бота на Python, настраивает webhook, деплоит на сервер*
💬 Пример 3: Веб-приложение
Ты: "Создай приложение для учёта расходов. React, тёмная тема, графики." AI: *создаёт проект на Vite + React, компоненты, графики на Recharts, localStorage*
💬 Пример 4: Анализ
Ты: "Открой avito.ru, найди квартиры во Владикавказе до 3 млн, составь таблицу" AI: *открывает браузер, парсит данные, создаёт таблицу*
