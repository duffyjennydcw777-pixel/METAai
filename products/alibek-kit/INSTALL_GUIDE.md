# 🚀 Полная Установка Среды Вайбкодинга

> Пошаговая инструкция от дяди Георгия. Следуй шагам по порядку.
> Время: ~30 минут.

---

## Шаг 0: Что тебе понадобится

- Компьютер на **Windows 10/11**
- Интернет
- Аккаунт Google (для Gemini)
- Аккаунт GitHub (бесплатно)

---

## Шаг 1: Установи VS Code (5 минут)

**VS Code** — это редактор кода, в котором мы будем работать.

1. Скачай: https://code.visualstudio.com/
2. Установи (всё по умолчанию, но поставь галочку **"Add to PATH"**)
3. Запусти VS Code

---

## Шаг 2: Установи Node.js (3 минуты)

**Node.js** — нужен для MCP серверов (расширения AI).

1. Скачай **LTS версию**: https://nodejs.org/
2. Установи (всё по умолчанию)
3. Проверь в терминале (Win+R → `cmd`):
   ```
   node --version
   npm --version
   ```
   Должны показать версии (например `v22.x.x` и `10.x.x`).

---

## Шаг 3: Установи Python (3 минуты)

**Python** — пригодится для скриптов и бэкенда.

1. Скачай **3.12+**: https://python.org/downloads/
2. ⚠️ **ОБЯЗАТЕЛЬНО** поставь галочку **"Add Python to PATH"** при установке!
3. Проверь:
   ```
   python --version
   ```

---

## Шаг 4: Установи Git (3 минуты)

**Git** — система контроля версий.

1. Скачай: https://git-scm.com/downloads
2. Установи (всё по умолчанию)
3. Настрой имя:
   ```
   git config --global user.name "Alibek"
   git config --global user.email "твой@email.com"
   ```

---

## Шаг 5: Установи Gemini Code Assist (5 минут) ⭐

**Это главное** — AI-ассистент, который будет писать код вместе с тобой.

1. В VS Code нажми `Ctrl+Shift+X` (расширения)
2. Найди **"Gemini Code Assist"** от Google
3. Установи
4. Перезагрузи VS Code
5. Войди в свой Google аккаунт когда попросит
6. Слева появится иконка Gemini — нажми на неё

> 💡 Gemini Code Assist работает **бесплатно** с ограничениями.
> Для полного доступа нужна подписка Google One AI Premium ($20/мес)
> или Gemini Business/Enterprise.

---

## Шаг 6: Установи MCP серверы (10 минут) ⭐⭐

**MCP (Model Context Protocol)** — это расширения, которые дают AI суперсилы:
документация в реальном времени, управление GitHub, и другое.

Открой терминал (в VS Code: `` Ctrl+` ``) и выполни:

```bash
# Context7 — доступ к документации любой библиотеки
npm install -g @context7/mcp

# Shadcn — компоненты UI
npm install -g shadcn

# Sequential Thinking — глубокий анализ проблем
npm install -g @anthropic/mcp-server-sequential-thinking

# GitHub — управление репозиториями из AI
npm install -g @anthropic/mcp-server-github
```

### Настройка MCP

1. Открой папку `C:\Users\ТВОЁ_ИМЯ\.gemini\antigravity\` (создай если нет)
2. Создай файл `mcp_config.json` с содержимым из `configs/mcp_config_template.json` (есть в этом пакете)
3. **Замени пути** на свои (вместо `ТВОЁ_ИМЯ` — твоё имя пользователя Windows)
4. Для GitHub MCP — создай Personal Access Token:
   - https://github.com/settings/tokens → "Generate new token (classic)"
   - Дай права: `repo`, `read:user`
   - Скопируй токен в `mcp_config.json`

---

## Шаг 7: Настрой правила AI (5 минут) ⭐⭐⭐

Это **самое важное** — правила определяют, как AI будет с тобой работать.

1. В корне папки `C:\Users\ТВОЁ_ИМЯ\.gemini\` создай файл **`GEMINI.md`**
2. Скопируй содержимое из `configs/GEMINI.md` (есть в этом пакете)
3. **Отредактируй под себя** — замени имя, стиль общения

### Для каждого нового проекта:
1. Создай папку `.agent/rules/` внутри проекта
2. Положи туда файлы из `configs/agent-rules/`
3. Заполни `MAKER_PROFILE.md` — это самый важный файл!

---

## Шаг 8: Создай первый проект (5 минут)

```bash
# Создай папку
mkdir C:\Dev\my-first-project
cd C:\Dev\my-first-project

# Инициализируй git
git init

# Создай структуру
mkdir .agent\rules
```

Скопируй файлы из `configs/agent-rules/` в `.agent/rules/`.

Теперь открой проект в VS Code:
```bash
code .
```

## Шаг 9: Проверка

1. Открой чат Gemini (иконка слева)
2. Напиши: **"Привет! Какие у тебя есть MCP инструменты?"**
3. AI должен рассказать про context7, shadcn, github и т.д.
4. Напиши: **"Прочитай правила из .agent/rules/ и расскажи мой профиль"**
5. Если AI отвечает на основе твоих файлов — **всё работает! ✅**

## ❓ Если что-то не работает

| Проблема | Решение |
|----------|---------|
| MCP не подключается | Проверь пути в `mcp_config.json`. Перезагрузи VS Code. |
| Gemini не отвечает | Проверь интернет. Перелогинься в Google. |
| Node.js не найден | Перезагрузи компьютер после установки. |
| Python не найден | Переустанови с галочкой "Add to PATH". |
