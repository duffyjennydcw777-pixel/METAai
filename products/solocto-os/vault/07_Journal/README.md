# 📓 07_Journal

> Daily/weekly reflections. Raw thoughts. Not structured — just write.

## Format
Each entry is a separate file: `YYYY-MM-DD_topic.md`

## Template
```markdown
---
date: YYYY-MM-DD
tags: [journal]
mood: [😤/😐/😊/🔥]
---

# [DATE] — [Title]

## What happened
[Free-form notes]

## What I learned
[Insights, realizations]

## Tomorrow
[Plan for next session]
```
