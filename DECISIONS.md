## DEC-001: Архитектура мульти-агентной системы
- **Дата**: 2026-04-28
- **Контекст**: Выбор инструмента для оркестрации AI-агентов
- **Варианты**: Raw OpenRouter API vs CrewAI vs LangGraph vs AutoGen
- **Решение**: Raw OpenRouter API (httpx + async)
- **Причина**: Минимум зависимостей, полный контроль, работает за 30 минут. CrewAI — на будущее.
- **Пересмотреть**: Когда pipeline станет >5 агентов или нужен stateful граф

## DEC-002: Модели для агентов
- **Дата**: 2026-04-28
- **Контекст**: Какие модели для каких задач
- **Решение**:
  - Architect/Security: Claude Sonnet 4 (умный, надёжный)
  - Review: Claude Haiku 3.5 (быстрый, дешёвый, достаточно умный)
  - Test Gen: GPT-4o-mini (дешёвый для рутинной генерации)
  - Preflight: Gemini Flash 2.0 (самый дешёвый, для простых проверок)
- **Причина**: Баланс стоимости и качества. $15-30/мес за всю армию.
- **Пересмотреть**: При выходе новых моделей или изменении цен

## DEC-003: Code Complexity Levels — 3 уровня
- **Дата**: 2026-04-28
- **Контекст**: Как определить объём тестирования для изменений
- **Решение**: 3 уровня (Trivial → Standard → Complex)
- **Причина**: Risk-based QA — тестировать пропорционально риску, не тратить время на тривиальные изменения
- **Пересмотреть**: Если обнаружатся баги, проскользнувшие через Level 1
