❗ Проблемы:
- Молчаливое игнорирование ошибок загрузки
- Возврат пустых строк без валидации
- Прямое использование переменных окружения

✅ Безопасная версия:
```python
def get_telegram_creds():
    try:
        from dotenv import load_dotenv
        env_path = Path(__file__).parent.parent / ".env"
        
        if not env_path.exists():
            logging.error(f"Env file not found: {env_path}")
            return None, None
        
        load_dotenv(env_path)
        
        token = os.environ.get(TELEGRAM_BOT_TOKEN_ENV)
        chat_id = os.environ.get(TELEGRAM_CHAT_ID_ENV)
        
        if not token or not chat_id:
            logging.warning("Incomplete Telegram credentials")
            return None, None
        
        return token, chat_id
    
    except Exception as e:
        logging.error(f"Error loading Telegram credentials: {e}")
        return None, None
```
