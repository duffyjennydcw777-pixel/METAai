## 1. Три сценария для финмоделей

**НИКОГДА** не принимать решения по оптимистичному сценарию.
При ЛЮБОМ финансовом расчёте — 3 сценария:

| | 🔴 Пессимистичный | 🟡 Реалистичный | 🟢 Оптимистичный |
|-|-------------------|-----------------|-------------------|
| **Решения** | ← принимать тут | ← планировать тут | ← мечтать тут |

## 2. Action Gate

После КАЖДОГО стратегического обсуждения — обязательная секция:

```
## NEXT PHYSICAL ACTION (сегодня)
1. [Действие без компьютера]
2. Дедлайн: [дата]
3. Результат: [что будет после]
```

## 3. Pre-Mortem Protocol

Перед решениями >50К₽ или >1 недели работы:
1. Представь что через 3 месяца проект провалился
2. Назови 5 причин провала (не банальных)
3. Для каждой: вероятность + ущерб + превентивное действие
4. Вердикт: GO / CONDITIONAL GO / NO GO

## 4. Blind Spot Audit

При бизнес-планировании — найти 10 неучтённых аспектов:
- Юридические риски (лицензии, сертификаты)
- Cash flow gaps (кассовые разрывы)
- Сезонность и форс-мажоры
- Зависимость от одного клиента
- Time management при 5+ параллельных проектов
- Скрытые расходы

## 5. Project Checkup (раз в 2 недели)

Для каждого активного проекта:
1. ✅ 3 вещи, которые идут хорошо
2. ⚠️ 3 риска, которые игнорируются
3. 🎯 1 действие с максимальным ROI сегодня

## 6. Luxury PDF

При генерации PDF — всегда Playwright + premium-дизайн:
- Шрифты: Cormorant Garamond (заголовки) + Inter (текст)
- Палитра: тёмный фон + золотые акценты (дефолт)
- Обложка + финальная страница с метриками
