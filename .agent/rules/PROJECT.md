## 🏗️ Архитектурные решения (ADR)

> Записывать ВСЕ значимые решения с обоснованием "Почему".

### ADR-001: Инициализация структуры
- **Решение**: Создана стандартная структура `.agent/rules/` с глобальными и проектными правилами
- **Обоснование**: Единообразие со всеми проектами (ONYX, Sylectus, Profit Pulse)
- **Дата**: 2026-04-26

### ADR-002: Raw OpenRouter API для агентов
- **Решение**: Использовать httpx + async вместо CrewAI/LangGraph
- **Обоснование**: Минимум зависимостей, полный контроль, работает за 30 минут
- **Дата**: 2026-04-28

### ADR-003: 5 ролей агентов (Architect/Review/Security/Test/Preflight)
- **Решение**: Разные модели для разных задач (Sonnet/Haiku/Mini/Flash)
- **Обоснование**: Баланс стоимости и качества. $15-30/мес за всю армию.
- **Дата**: 2026-04-28
